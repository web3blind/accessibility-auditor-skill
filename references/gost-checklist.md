## Расширенный чеклист из статьи (практическая проверка)

### Клавиатурная навигация
- [ ] Весь функционал сайта доступен с клавиатуры
- [ ] Нет ограничений по времени нажатия клавиш
- [ ] Порядок фокуса логичен и соответствует смыслу страницы
- [ ] Структура страницы программно определяема (корректная семантика)
- [ ] Сайт корректно работает с программами экранного доступа

### Масштабирование
- [ ] Текст масштабируется минимум до 200% без потери функциональности
- [ ] При увеличении текста отсутствует горизонтальная прокрутка
- [ ] Интерфейс сохраняет управляемость при пользовательских настройках масштаба

### PDF и документы
- [ ] PDF-документы размечены по требованиям ГОСТ Р 70176-2022
- [ ] Для недоступных PDF размещена альтернативная версия
- [ ] Не используются сканированные изображения документов без текстового слоя

### Изображения и медиа
- [ ] Все изображения имеют содержательные альтернативные тексты
- [ ] Диаграммы и графики имеют текстовое представление данных
- [ ] Иконки и элементы управления имеют доступные названия
- [ ] Декоративные элементы скрыты от вспомогательных технологий
- [ ] Видео/аудио сопровождаются текстовым описанием
- [ ] CAPTCHA имеет альтернативный доступный способ подтверждения

### Контраст и цвет
- [ ] Контраст текста и фона не ниже 4,5:1 (крупный текст — 3:1)
- [ ] Цвет не используется как единственный способ передачи информации
- [ ] Используются читаемые шрифты без избыточной декоративности

### Навигация и ссылки
- [ ] Заголовки страниц отражают их тему или цель
- [ ] Текст ссылок понятен вне контекста
- [ ] Навигация последовательна и предсказуема
- [ ] Элементы интерфейса при наведении не изменяют контекст автоматически

### Формы
- [ ] Каждое поле формы имеет текстовую метку
- [ ] Обязательные поля обозначены текстом
- [ ] При ошибке выводится текстовое пояснение
- [ ] Сообщения об ошибках доступны программам экранного доступа
- [ ] Инструкции по заполнению форм представлены текстом

### Движение и анимация
- [ ] Пользователь может остановить/скрыть движущийся контент
- [ ] Отсутствуют мигающие элементы более 3 вспышек в секунду
- [ ] Анимация используется только при функциональной необходимости

### Соответствие стандартам
- [ ] Проведен внутренний аудит на соответствие ГОСТ Р 52872-2019

## ГОСТ Р 52872-2019 — Ключевые положения

ГОСТ Р 52872-2019 «Интернет-ресурсы и другая информация, представленная в электронно-цифровой форме. Приложения для стационарных и мобильных устройств, иные пользовательские интерфейсы. Требования доступности для людей с инвалидностью и других лиц с ограничениями жизнедеятельности» — это российский национальный стандарт, базирующийся на WCAG 2.1.

Основные разделы:
1. Воспринимаемость — вся информация должна быть доступна пользователям
2. Управляемость — интерфейс должен быть управляемым
3. Понятность — информация и интерфейс понятны
4. Совместимость — максимальная совместимость с вспомогательными технологиями

Уровни соответствия (аналогично WCAG):
- Уровень А — базовые требования
- Уровень АА — рекомендуемый уровень для госсайтов
- Уровень АAА — расширенный уровень

**Для госсайтов обязателен уровень АА.**
